## 🎯 Visão Geral

Este é um portfólio web completo, moderno e extremamente visual que apresenta a trajetória profissional, projetos técnicos, certificações e habilidades de **Danilo Adryel**, um profissional especializado em:

- ⚙️ **Qualidade Automotiva** (Infotainment & E/E Systems)
- 🔌 **Sistemas Embarcados** (ESP32, STM32, RTOS)
- 📊 **Análise de Dados** (Power BI, DAX, KPIs)

### 🎨 Design Highlights

- **Interface Fluida**: Animações suaves e transições elegantes
- **Design Moderno**: Gradientes, glassmorphism e sombras sofisticadas
- **Responsivo**: Adaptável para mobile, tablet e desktop
- **Dark Mode**: Toggle entre tema claro e escuro

## 🚀 Instalação e Uso

### Opção 1: Uso Direto
```bash
# Clone o repositório
git clone https://github.com/DARXS/portfolio.git

# Acesse a pasta
cd portfolio

# Abra o index.html no navegador
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

### Opção 2: Servidor Local
```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx http-server

# Acesse: http://localhost:8000
```

### Opção 3: VS Code Live Server
1. Instale a extensão "Live Server"
2. Clique direito em `index.html`
3. Selecione "Open with Live Server"

## 🎨 Personalização

### Alterar Cores
Edite as variáveis CSS em `css/style.css`:

```css
:root {
    --primary-color: #3b82f6;    /* Azul principal */
    --accent-color: #f59e0b;     /* Laranja destaque */
    /* ... outras variáveis */
}
```

### Alterar Conteúdo
Edite `index.html` diretamente nas seções desejadas.

### Integrar com Outras APIs
Adicione novas funções em `js/main.js`:

```javascript
async function fetchCustomAPI() {
    const response = await fetch('YOUR_API_URL');
    const data = await response.json();
    // Process data
}
```
